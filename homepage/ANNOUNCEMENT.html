Announcement: GNU philosophy 0.1.20140214

The GNU project is pleased to announce GNU philosophy 0.1.20140214 released February 14, 2014. This is a stable release and the first official GNU release.

A big thank you to Richard M. Stallman, the main author of this package.

News

0.1

Description

GNU philosophy is the GNU Project’s exhaustive collection of articles and essays about free software and related matters. It provides all of them standalone or collected as a huge single book, in Info, HTML, PDF, plain text and every other format supported by Texinfo. It makes the individual Texinfo source code of any essay readily available for inclusion in another package’s documentation and also implements a way to generate the whole GNU website philosophy section automatically from the Texinfo sources. Last, but not least, it facilitates the future work of the FSF (Free Software Foundation) in the publication of free software articles collection books like: “Free Software, Free Society: Selected Essays of Richard M. Stallman”.
