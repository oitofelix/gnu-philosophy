Manual

The documentation is included in the distribution tarball that you can find in Download, but as it can be useful to obtain it separately, notably because of its reference material, we make it available standalone in several formats optionally compressed. The following formats are available:

  • HTML (1184K bytes) - entirely on one web page.
  • HTML - with one web page per node.
  • HTML compressed (272K gzipped characters) - entirely on one web page.
  • HTML compressed (384K gzipped tar file) - with one web page per node.
  • Info document (240K bytes gzipped tar file).
  • ASCII text (832K bytes).
  • ASCII text compressed (236K bytes gzipped).
  • TeX dvi file (392K bytes gzipped).
  • PDF file (1152K bytes).
  • Texinfo source (8K bytes gzipped tar file).

You can buy printed copies of some manuals (among other items) from the Free Software Foundation; this helps support FSF activities.
