2.4 Authors

This chapter contains a list of those people who are the writers of the philosophical essays and articles found herein. This program would not be what it is today without the invaluable help of these people, to whom we would like to say:

THANK YOU VERY MUCH!

The names are listed in chronological order of contribution. Each person’s contact information, usually email, and a summary of that person’s contribution is put in per correspondent section.

For the list of maintainers of this package through the ages, see Maintainers.

For other external contributions, that does not matter for copyright purposes, e.g., bug reports, patch for markup and good ideas, see Thanks.

If you would appreciate your own name listed here, please contribute! See Contributing.
